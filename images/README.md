# Изображения

Сейчас сайт показывает **временные SVG-заглушки** в фирменных цветах.
Чтобы поставить реальные фото — просто положите сюда файлы с этими именами
(формат `.jpg`). Код подхватит их автоматически, ничего менять не нужно.

| Файл | Где используется | Рекомендуемый размер |
|------|------------------|----------------------|
| `hero.jpg` | Полноэкранный фон Hero | 1920×1080, горизонт. |
| `about.jpg` | Фото в секции «О нас» | 900×1100, вертикальное |
| `gallery1.jpg … gallery5.jpg` | Слайдер атмосферы | 900×700, горизонт. |
| `dish1.jpg`, `dish2.jpg`, `dish3.jpg` | Карточки блюд | 800×600, горизонт. |
| `menu1.jpg`, `menu2.jpg`, `menu3.jpg` | Страницы меню в боковой панели | вертикальные, любая высота |

Заглушки (`placeholder-*.svg`) можно не трогать — они служат запасным
вариантом, если соответствующий `.jpg` ещё не загружен. Скрипт
`_generate_placeholders.py` пересоздаёт их при необходимости.
